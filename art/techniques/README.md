# Technique icon sources

The PNG files in `generated/` are the high-resolution source artwork for the
radial Technique wheel. Runtime textures are downscaled to 64×64 under
`src/main/resources/assets/onepunchcrafts/textures/gui/techniques`.

The accepted direction is a black-and-gold fighting-game emblem with restrained
red and white accents. Serious Punch intentionally uses the contrasting manga-ink
direction selected for that Technique.
